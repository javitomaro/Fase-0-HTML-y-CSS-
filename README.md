# 🚀 Proyecto Phoenix Framework - Aprendizaje HTML & CSS

Este repositorio contiene mi primer proyecto desarrollado con el framework **Phoenix (Elixir)**. El objetivo principal de este trabajo es dominar la estructura de directorios de una aplicación profesional y la implementación de interfaces modernas utilizando HTML5 y CSS3.

## 🛠️ Tecnologías utilizadas
* **Phoenix Framework:** Motor de plantillas y estructura backend.
* **Elixir:** Lenguaje base del proyecto.
* **HTML5 & CSS3:** Estructura y diseño visual.
* **Git:** Control de versiones.

## 📈 Objetivos del Proyecto
1. Comprender el flujo de renderizado de vistas en Phoenix.
2. Aplicar estilos CSS modulares y responsivos.
3. Organizar el código siguiendo las mejores prácticas de la arquitectura MVC.
